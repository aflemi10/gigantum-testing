# Gigantum Testing
Automation of Gigantum testing with Selenium

### Getting ChromeDriver (Google Chrome)
Using MacOS 

```bash
brew install chromedriver
```
### Getting geckodriver (Mozilla Firefox)
Using MacOS

```bash
brew install geckodriver
```
